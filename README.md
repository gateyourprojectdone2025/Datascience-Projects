## Salary Prediction Based on Country and Race Using Data Science
## Overview

-This project aims to predict annual salary based on demographic and socio-economic features such as country, race, education level, occupation, age, and other attributes. By applying machine learning techniques, the model learns patterns in income distribution and helps understand how demographic factors influence salary.
-This project highlights data preprocessing, feature encoding, exploratory data analysis, model training, and evaluation of regression  classification models depending on prediction needs.

## Technologies Used

-Python

-NumPy, Pandas

-Matplotlib / Seaborn

-Scikit-learn

-Jupyter Notebook / Google Colab
