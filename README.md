## Room Occupancy Detection Using Data Science
## Overview

Room Occupancy Detection is a machine learning-based project designed to determine whether a room is currently occupied using environmental sensor data. It optimizes energy usage by automatically managing devices like lights and air conditioning when the room is empty, improving cost efficiency in smart buildings. The model analyzes data such as Temperature, Humidity, CO₂ Levels, and Light Intensity to predict occupancy accurately.


## Features

-Detects room occupancy using real-time sensor data

-Uses machine learning classification models

-High accuracy prediction using Random Forest Algorithm

-Helps reduce power and energy consumption

-Can be integrated with IoT devices like ESP32 / Raspberry Pi

-Scalable for Smart Homes, Offices, Colleges, and Industries


## Technologies Used
-Python

-Pandas, NumPy

-Scikit-Learn

-Matplotlib / Seaborn

-Jupyter Notebook / VS Code

## Advantages

-Reduces unnecessary energy consumption.

-Helps automate lighting and air conditioning.

-Improves operational efficiency in large buildings.

-More cost-effective compared to manual checking.

-Can run in real-time with IoT integration.

## Disadvantages

-Requires sensors to be installed, which adds hardware cost.

-Sensor noise or faulty readings may affect accuracy.

-Works best in controlled indoor environments only.

-Needs periodic calibration for long-term performance.

## Applications

-Classroom and Office Energy Management
